That "series" of booleans has the same length as the number of rows. For any row where the age is greater than 20, it will be marked as True; otherwise, it will be False.
```python
D[ D["Age"] > 25 ]
```

```python
df = pd.DataFrame([
    ['red', 255, 0, 0],
    ['blue', 0, 0, 255],
    ['green', 0, 255, 0],
    ['purple', 127, 0, 255],
])
df.columns=['name', 'R', 'G', 'B']
print(df)
```
